Firstly download JavaFx from ```https://gluonhq.com/products/javafx/```

Secondly open ```run_WarAboveUs.bat``` in edit mode.
Set module-path to lib folder of downloaded JavaFx. Example:
```--module-path "E:\Programy\Java\javafx-sdk-21.0.5\lib"```
Save and exit

Finally run the application by executing run_WarAboveUs.bat.
